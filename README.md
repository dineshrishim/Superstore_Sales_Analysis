# Superstore Sales Analysis

Analyzed $2.3M in Superstore sales using Excel pivot tables and charts.

## Project Highlights
- Explored trends by product category, region, month, and top customers
- Produced actionable recommendations for inventory, marketing, and customer retention
- Created a dashboard summarizing insights with clear visualizations

## Files in this Repository
- `Superstore_Sales_Analysis.xlsx` → Full dataset + pivot tables + dashboard
- `Superstore_Sales_Dashboard.pdf` → Clean dashboard for quick viewing
